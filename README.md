# Planify — Task Management Application

A clean and functional task manager built with Vanilla JavaScript. Helps users organize their daily tasks with a modern, responsive interface.

![Screenshot](design.png)

## ✨ Features

- **Full CRUD operations**: Create, read, update (complete), and delete tasks
- **Real-time search** with live filtering
- **Data persistence** using LocalStorage
- **Smooth animations** when deleting tasks
- **Responsive design** — works perfectly on mobile and desktop
- **Clean OOP architecture** using ES6+ classes
- **Keyboard friendly** and accessible UI

## 🛠 Technologies

- HTML5
- SCSS (modular, with custom properties and BEM-style)
- Vanilla JavaScript (ES6+)
- LocalStorage

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Planify.git
   ```

2. Open the project folder and run `index.html` (or use Live Server in VS Code).

3. Start adding tasks!
